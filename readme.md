# Gamplo API Plugin for Godot v4.6.x
Thank you to @AwsmeAnthony he made his own [Gamplo Plugin](https://github.com/AwsmeAnthony/Gamplo-Godot-Plugin), I didn't know how to get the token but his repository helped a bunch.

I didn't really like the format so I made my own, check his out if you want to see it. (Not sure what Godot Version it's usable for)

## Features
- [x] Auto login if your logged in
- [x] Works offline without errors
- [x] Get all of your game's Achievements & Unlock specific Achievements
